# vimrc
Simple vim configuration

INSTALL
----------------

1. Checkout from github

	git clone git://github.com/smicyk/vimrc.git ~/.vim
	cd ~/.vim
	git submodule update --init

2. Install vimrc

	./install-vimrc.sh

3. Command-T installation

	cd .vim/bundle/command-t/ruby/command-t
	ruby extconf.rb
	make
